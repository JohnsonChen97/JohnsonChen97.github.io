The Project

For one of my classes in the second year of my program, I was given an assignment to build a mobile application for our campus' room rental website. The website had many problems that were blatant. The more obvious ones include lots of wasted space, unintuitive buttons, and a poor UI. This made for an unpleasant user experience.

The home/landing page of the website.

The edit reservation page.

My solution for the app involves creating an overall better flow. Users will be able to access many parts of the system without having to press the back key multiple times. Creating a more streamlined approach for the main purpose of the app; renting rooms, is easily done and simpler to understand with more editing abilities. The entire UI has been revamped and extra features have been added such as notifications. There is little emphasis on text and more on making sure the user gets what they need. These changes fix the overall problem by giving the user an application they can use to reliably book rooms while having the ability to come back and review what they last did.
