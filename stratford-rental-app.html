Stratford Campus - App Design

TOOLS

ROLE

DURATION

March 2018 - April 2018

Figma, Adobe Illustrator, InVision

UI/UX Designer

Overview

Stratford campus' project room checkout website did not have an app to go along with it. The website is also outdated and needs a better way to book rooms.

The Project

Design an app with updated UI/UX that allows students to make reservations easily.

The main problems:

  • Lots of wasted/blank space.
  • Unintuitive buttons.
  • Poor user experience.

My Solution

Create an overall better flow. Users will be able to access many parts of the system without having to press the back key multiple times.

Creating a more streamlined approach for the main purpose of the app; renting rooms, is easily done and simpler to understand with more editing abilities.

The entire UI has been revamped and extra features have been added such as notifications. There is little emphasis on text and more on making sure the user gets what they need.
